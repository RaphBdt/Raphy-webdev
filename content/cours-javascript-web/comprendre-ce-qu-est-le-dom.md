**DOM = Document Object Model.** Il s'agit d'une interface de programmation qui est une représentation du HTML et qui permet d'accéder aux éléments de la page et de les modifier avec JavaScript.

Dans le DOM, on commence par un élément de racine qui est le point de départ. Par exemple, la balise _HTML_ a deux enfants : _head_ et _body_. Ces deux éléments ont donc un parent commun : _html_.

## Utilité du DOM

Grâce à l'interface de programmation qui nous permet de parcourir le DOM, on peut intéragir avec lui :

* Modifier du contenu
* Modifier style d'un élément
* Création / suppression d'un élément
* Intéraction avec les utilisateurs (repérer des clics sur un élément, récupérer leur nom dans un formulaire...)
* ...
